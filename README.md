# 📊 Student Performance Report (Excel Project)

## 📌 Description

This project is created using Microsoft Excel to manage student performance data.

## 🚀 Features

* Result calculation using IF formula
* Grade system using nested IF
* Conditional formatting for Pass/Fail
* Data organized using Excel Table
* Chart for performance visualization

## 🛠 Tools Used

* Microsoft Excel
